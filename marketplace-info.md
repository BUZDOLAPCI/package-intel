# Dedalus Marketplace Info

## Short Description
Query npm, PyPI, and crates.io for package intelligence

## Long Description
Get package summaries with latest versions and download stats, view release timelines with version history, and analyze maintenance signals like commit frequency and issue response times. Tools: package_summary, release_timeline, maintenance_signals.
